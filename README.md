# Mini ecommerce MERN APP
Full Stack MERN ecommerce application featuring Reduc toolkit, JWT authentication, MongoDB Atlas, Cloudinary , and deployed on Netlify and Render.**

## Features:
### Admin
- Login and logout
- View products
- Add,edit,delete products
- Upload product images to Cloudinary
### User
- Sign up and login
- View products
- Search products
- Filter by category
- Sort by price
- Pagination
- Add products to cart


## Live Demo
https://ecommerce-demo-by-shaima.netlify.app/

## Tech Stack

### Frontend
- React
- Redux toolkit
- React Router
- Axios
- TailwindCSS
- Vite
- Netlify for deployment
### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Multer for file upload
- Cloudinary for asset storage
- Render for deployment
